\# Projet Bibliothèque - SAE Base de Données

\## Résumé

Ce projet modélise un système de gestion de bibliothèque permettant de gérer les livres, 

les auteurs, les éditeurs, les usagers ainsi que les emprunts et les retours.



\## Technologies

\- SGBD : SQL Server

\- Outils : SQL Server Management Studio (SSMS), draw.io



\## Structure du dépôt

\- `/sql` : Scripts SQL (création, insertion, requêtes)

\- `/data` : Données au format CSV

\- `/docs/diagrams` : Diagrammes MER, MLD, MPD

\- `/docs/screenshots` : Captures d'écran des résultats

\- `REPORT.pdf` : Rapport final 



\## Installation

1\. Exécuter `sql/00\_create\_schema.sql` pour créer les tables

2\. Exécuter `sql/10\_insert\_sample\_data.sql` pour insérer les données de test

3\. Exécuter `sql/20\_queries\_required.sql` pour les requêtes obligatoires
